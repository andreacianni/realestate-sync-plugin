Get All Agencies
Description
Retrieve a list of agencies with filtering options. This endpoint allows pagination, sorting, and filtering by taxonomy terms or metadata.

Request Method
POST

URL
https://residence.me/wp-json/wpresidence/v1/agencies﻿

Request Parameters
page

Type: integer
Required: No (Optional, defaults to 1)
Description: Controls pagination of results.
posts_per_page

Type: integer
Required: No (Optional, defaults to 10, max 100)
Description: Controls the number of results per page.
fields

Type: string
Required: No (Optional)
Description: Comma-separated list of fields to include in the response. Supports dot notation for nested fields.
taxonomies

Type: object
Required: No (Optional)
Description: Filters agencies by taxonomy terms, each taxonomy can contain multiple terms (OR relationship).
Available taxonomies:

property_city
property_area
property_category
property_action_category
meta

Type: object
Required: No (Optional)
Description: Filters agencies by meta fields. Each meta query requires value, compare, and type.
Compare operators:

=, !=, >, >=, <, <=
LIKE, NOT LIKE
IN, NOT IN
BETWEEN, NOT BETWEEN
EXISTS, NOT EXISTS
Available types:

NUMERIC, BINARY, CHAR, DATE, DATETIME, DECIMAL, SIGNED, TIME, UNSIGNED
Response Structure
status

Type: string
Values: "success" or "error"
Description: Indicates whether the request was successful.
query_args

Type: object
Description: Contains all parameters used in the query, useful for debugging and verification.
data

Type: array
Description: Contains an array of agency objects.
